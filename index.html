ControllerDll (Joypad BLE Mouse Control)

A controller-input plugin that turns supported joypads into a mouse-and-action surface for the TabDocks runtime.

Input Layer

HID-based controller input handling with support for PS3/PS4 style mappings and plugin runtime loading.

Build Output

Shipped as a plugin DLL package and loaded into the runtime plugin folder of the dock application.

  • Reads controller input through HID-style mappings suitable for gamepads and Bluetooth-connected pads.
  • Translates buttons and stick movements into pointer movement and desktop actions.
  • Works as an optional plugin so the dock can run without the controller layer loaded.
  • Useful for couch control, living-room use, and hands-off desktop navigation.

RemoteControl Stack (Passive HTTP Endpoints)

A remote-control system built around passive HTTP polling, a PHP endpoint layer, the Android remote client, and the RemoteControl.dll module.

Passive HTTP Endpoints

PHP endpoints: get.php, put.php, get_screen.php, put_screen.php for polling-style remote interactions.

Android Remote App

Companion remote interface project used to trigger actions and push commands to the desktop host.

RemoteControlDll

.NET module containing HTTP parser/transport, encryption helper, FTP manager, and related WPF config pages.

  • Passive request/response flow for devices that poll for state, commands, or screen data.
  • PHP endpoints handle get, put, and screen-transfer operations without requiring an always-on websocket server.
  • RemoteControl.dll handles parsing, encryption, FTP synchronization, and desktop control helpers.
  • The Android remote app provides a portable interface for triggering actions and managing the dock remotely.

ESP32 Daikin AC Controller

ESP32 firmware and Android app for controlling Daikin air conditioners via Tasmota IR adapter. Features Wi-Fi control AP, temperature-based routine scheduler, and REST API endpoints.

ESP32 Firmware

ESP-IDF based firmware with HTTP server, NVS storage, SNTP time sync, and routine scheduler with duration tracking.

Android Controller

Kotlin/Compose app with Config, Control, and Routine tabs for managing ESP32 and scheduling AC operations.

IR Communication

Connects to Tasmota IR Wi-Fi adapter to send IRHVAC commands for Daikin AC control via HTTP bridge.

  • ESP32 creates control AP (ESP32-DAIKIN) for direct Android connection without internet.
  • REST API for immediate commands, routine configuration, and status monitoring.
  • Temperature-based routine scheduler with 8 configurable routines and duration settings (5-1440 minutes).
  • Auto-transition between routines when duration expires, supporting temperature thresholds.
  • NVS persistence for Wi-Fi config and routine state across reboots.
  • ESP32 internal temperature sensor exposed for ambient monitoring.
  • SNTP time sync with fallback to manual epoch sync from Android app.
  • Android app provides Config tab (adapter setup, sensor data, time sync), Control tab (immediate AC commands), and Routine tab (scheduler management).
