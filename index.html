TabDocks

A customizable tabbed docking bar for Windows — quick-launch apps, manage windows, and control your desktop from anywhere.

⚡ Features

🗂️

Tabbed Dock Bar

Attach a multi-tab dock to any screen edge. Each tab groups related shortcuts and apps with its own color scheme.

🚀

Application Launcher

Launch any executable with custom arguments and optional elevated (admin) privileges directly from the dock.

🎨

Visual Customization

Per-tab colors, blur effects, transparency, multiple themes, and adjustable dock thickness.

🌐

Remote Control

HTTP/PHP-based interface for operating and configuring the dock from a remote device or Android app.

📡

FTP Sync

Bi-directional folder synchronisation between local and remote FTP servers with scheduled intervals.

🎮

Gamepad Support

Full PS3 and PS4 DualShock controller integration for navigating and triggering dock actions.

📺

Screen Streaming

Capture and stream the desktop display to a remote viewer over the network.

⌨️

Keyboard & Mouse Hooks

Raw input capture, custom hotkeys, and Copilot key remapping through low-level Windows hooks.

🔒

Secure Credentials

Remote-control credentials are encrypted before being stored in the INI configuration file.
