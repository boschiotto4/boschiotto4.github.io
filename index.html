TabDocks (C# Dock Bar)

A Windows dock bar built with .NET 8, WPF, and WinForms integration. It supports tabbed launchers, FTP sync, remote operations, and a built-in calendar popup.

Dock + Launcher

Screen-edge AppBar behavior, per-tab launch items, command arguments, and optional elevated execution.

Calendar With Week Numbers

Dedicated calendar window with a visible Week column, Monday as first day of week, and themed calendar colors.

Config + Packaging

Release packaging to site downloads with zip artifacts and manifest updates via release metadata JSON.

RemoteControl Stack (Passive HTTP Page)

Remote operation stack composed of a passive PHP HTTP page, Android remote interface, and RemoteControl.dll. Supports request/response control endpoints, FTP sync workflow, and optional screen-stream transport.

Passive HTTP Endpoints

PHP endpoints: get.php, put.php, get_screen.php, put_screen.php for polling-style remote interactions.
